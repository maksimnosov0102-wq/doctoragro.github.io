# doctoragro.github.io
AgroDoctor AI – інформаційна платформа та інтелектуальний агрономічний асистент для підтримки прийняття рішень у технологіях захисту рослин. Сайт містить базу знань про хвороби, шкідників, засоби захисту рослин, системи вирощування сільськогосподарських культур та рекомендації на основі штучного інтелекту.
